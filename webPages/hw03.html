    2. Click the Settings tab in the top navigation bar.
    3. Enable GitHub Pages
    4. Scroll down to the "Code and automation" section in the left sidebar and click Pages.
    5. Under Build and deployment, set the Source dropdown menu to Deploy from a branch.
    6. Under Branch, choose your default branch (usually `main` or master) and keep the folder path as `/(root)`.

Работа с таблицами

Создайте таблицу (`table`), состоящую из 3 строк и 3 столбцов. В первой строке таблицы укажите заголовки (`th`) для каждого столбца (например, "Имя", "Возраст", "Город"). В каждой из остальных строк создайте ячейки (`td`), заполнив их произвольными данными (например, три имени, возраст и города).
