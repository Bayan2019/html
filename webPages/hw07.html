Единицы и цветовые модели в CSS

Работа с единицами измерения

  • Используйте px для задания точных размеров элементов.

    background-color: yellowgreen;
    width: 800px;
    height: 80px;

  • Примените em для задания относительных размеров шрифта внутри контейнера.

    background-color:yellow;
    font-size: 1.5em;
    margin: 8px;
    background-color: yellowgreen;
    width: 800px;
    height: 150px;
    font-size: 17px;
    display: inline-flex;

  • Используйте rem для задания размеров относительно корневого элемента (html).

    background-color: yellowgreen;
    width: 50rem;
    height: 80px;

  • Примените % для задания ширины контейнеров относительно их родителя.

    background-color:yellow;
    margin: 8px;
    width: 30%;
    background-color: yellowgreen;
    width: 800px;
    height: 150px;
    font-size: 17px;
    display: inline-flex;

Работа с цветовыми моделями

  • Задайте цвет текста и фона с помощью шестнадцатеричной системы (hex)

    background-color: #92FF92;
    width: 800px;
    height: 80px;

  • Используйте модель rgb для задания цветов фона и текста.

    background-color: rgb(146,255,146);
    width: 800px;
    height: 80px;

  • Примените модель rgba для добавления прозрачности к цветам.

    background-color: rgba(0,0,255, 0.3);
    width: 800px;
    height: 80px;
