Селекторы

Работа с базовыми селекторами

  1. Селектор по тегу (например, все абзацы должны иметь синий цвет текста).

  2. Селектор по классу (например, элементы списка с определённым классом должны быть жирными).

    • oranges
    • lemons
    • grapefruits
    • limes
    • kiwis

  3. Селектор по ID (например, заголовок с определённым ID должен быть красного цвета).

    Заголовок

Приоритет селекторов

  1. Создайте несколько конфликтующих стилей для одного элемента с использованием селекторов тегов, классов и ID (классы важнее тегов).

  2. Покажите, какой стиль применится в итоге, объяснив принцип приоритета (ID важнее классов).

Если !important нет, браузер вычисляет «вес» селектора по системе баллов: от тысяч до единиц (1000 - 100 - 10 - 1). Чем больше вес, тем стиль важнее.

Специфичность оценивается по четырем разрядам:
  1. 1-0-0-0 — инлайновые стили в атрибуте style.
  2. 0-1-0-0 — идентификаторы (#id).
  3. 0-0-1-0 — классы (.class), атрибуты ([type="text"]) и псевдоклассы (:hover, :focus).
  4. 0-0-0-1 — теги (div, p, span) и псевдоэлементы (::before, ::after).
